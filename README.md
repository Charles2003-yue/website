# 智能路径规划：旅行商问题研究展示网站

## 项目简介

这是一个视觉效果炸裂的网站，用于展示学术论文《Bandit based Dynamic Candidate Edge Selection in Solving Traveling Salesman Problems》的研究内容。网站使用通俗易懂的中文语言，让非技术背景的用户也能理解这项前沿研究的要点和亮点。

## 论文信息

- **标题**: Bandit based Dynamic Candidate Edge Selection in Solving Traveling Salesman Problems
- **中文标题**: 基于多臂老虎机的动态候选边选择求解旅行商问题
- **作者**: Long Wang, Jiongzhi Zheng, Zhengda Xiong, ChuMin Li, Kun He
- **机构**: 华中科技大学计算机科学与技术学院，法国皮卡第大学

## 研究亮点

### 🎯 核心创新
- 引入多臂老虎机（MAB）模型来动态选择候选边
- 提出三种智能选择策略：ε-贪婪、M-贪婪、α-贪婪
- 实现了自适应的候选边选择机制

### 📊 实验成果
- 在82个TSPLIB基准测试中表现优异
- 平均解质量提升57%
- 成功率在33/62个测试实例中超越传统LKH算法
- 扩展到LKH-3，在MTSP和CVRP问题上也取得显著提升

### 🌍 实际应用
- 物流配送路线优化
- 芯片制造路径规划
- 旅游景点游览规划
- 车辆调度优化
